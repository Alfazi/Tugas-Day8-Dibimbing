# Kesimpulan & Pandangan Penggunaan GitHub dalam Tim

GitHub merupakan alat yang sangat membantu dalam pengembangan proyek secara kolaboratif.  
Dengan fitur seperti branching, pull request, dan review, setiap anggota tim dapat bekerja secara paralel tanpa mengganggu pekerjaan anggota lain.  

Selain itu, GitHub juga menyediakan version control yang memudahkan pelacakan perubahan kode, serta memungkinkan rollback jika terjadi kesalahan.  
Proses code review membantu meningkatkan kualitas kode dan memastikan standar yang konsisten di seluruh tim.

Secara keseluruhan, penggunaan GitHub menjadikan proses pengembangan proyek lebih terstruktur, aman, dan efisien.
